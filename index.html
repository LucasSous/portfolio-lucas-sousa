Sobre Mim

Sou desenvolvedor Web, técnico em informática e atualmente estou cursando Ciências da Computação na instituição Unifametro. Possuo conhecimentos básicos em tecnologias de frontend e backend como Vue js e Boot Strap. Sou uma pessoa muito focada em objetivos, sempre busco melhorias tanto para somar e ajudar a quem está ao meu redor como para benefícios profissionais.

Projetos
Livraria - Frente de Loja (Aplicativo Web - Em andamento!)

Sistema que estou incluso, sendo um dos desenvolvedores web, atuando tanto no backend como no frontend e utilizando os frameworks SpringBoot e Vue JS. Trata-se de um sistema que visa gerenciar uma livraria, contendo cruds bem específicos, controle de acesso e regras de negócios bastante complexas. Tem sido um ótimo projeto, pois me ajudou muito a conhecer e acompanhar de perto como é desenvolvido um sistema profissional e me especializar ainda mais nesse imenso mundo da programação e gerenciamento de projetos.

Locadora de Livros (WEB)

Sistema de locação de livros onde o principal intuito é o controle e gerenciamento de alugueis, nesse sistema existem alguns cruds, como de clientes, livros, editoras e alugueis, também existe uma dashboard na tela principal para ter uma visão geral do sistema. O sistema foi desenvolvido com os frameworks SpringBoot e Vue JS e também foi utilizado o framework CSS Vuetify para estilização das telas.

Locadora de Livros (Mobile)

Nesse sistema foi utilizado a mesma api que foi utilizada na locadora de livros (web), tendo o mesmo intuito e as mesmas funcionalidades, porém esse sistema foi desenvolvido com um framework Flutter, utilizado para desenvolvimentos de aplicações mobile. O sistema ainda esta em desenvolvimento!
