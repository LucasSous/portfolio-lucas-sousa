Sobre Mim

Sou desenvolvedor front end, técnico em informática e faço o curso de Ciências da Computação na instituição Unifametro. Possuo conhecimentos básicos em tecnologias de frontend e backend como Vue js, React js, Boot Strap, etc. Sou uma pessoa muito focada em objetivos, sempre busco melhorias tanto para somar e ajudar a quem está ao meu redor como para benefícios profissionais.

Projetos
Livraria - Frente de Loja (Aplicativo Web - Em andamento!)

Sistema que eu estava incluso, sendo um dos desenvolvedores web, atuando tanto no backend como no frontend e utilizando os frameworks SpringBoot e Vue JS. Trata-se de um sistema que visa gerenciar uma livraria, contendo cruds bem específicos, controle de acesso e regras de negócios bastante complexas. Foi um ótimo projeto, pois me ajudou muito a conhecer e acompanhar de perto como é desenvolvido um sistema profissional e me especializar ainda mais nesse imenso mundo da programação e gerenciamento de projetos.

Locadora de Livros (WEB)

Sistema de locação de livros onde o principal intuito é o controle e gerenciamento de alugueis, nesse sistema existem alguns cruds, como de clientes, livros, editoras e alugueis, também existe uma dashboard na tela principal para ter uma visão geral do sistema. O sistema foi desenvolvido com os frameworks SpringBoot e Vue JS e também foi utilizado o framework CSS Vuetify para estilização das telas. Obs: Este projeto também foi feito com o React como treinamento para um projeto que fui alocado na empresa em que trabalho.

Locadora de Livros (Mobile)

Nesse sistema foi utilizado a mesma api que foi utilizada na locadora de livros (web), tendo o mesmo intuito e as mesmas funcionalidades, porém esse sistema foi desenvolvido com um framework Flutter, utilizado para desenvolvimentos de aplicações mobile. O sistema ainda esta em desenvolvimento!

Habilidades

HTML é uma abreviação de Hypertext Markup Language, ou seja, Linguagem de Marcação de Hipertexto. Resumindo, o HTML é uma linguagem usada para a publicação de conteúdo (texto, imagens, vídeos, áudio etc.) na web.

CSS é uma linguagem de estilo, também conhecida como folhas de estilo em cascata. É usada para personalização visual de um site.

JavaScript é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe, mais conhecida como a linguagem de script para páginas Web.

Em poucas palavras, é uma biblioteca JavaScript para criação de interfaces para o usuário, desenvolvida e mantida pelo Facebook.

O Vue.js (comumente conhecido como Vue, pronunciado "view") é um framework progressivo do JavaScript de código aberto (open source) para a construção de interfaces de usuário.
